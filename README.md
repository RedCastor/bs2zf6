Bootstrap 3 to Zurb Foundation 6
=============================

[Bootstrap][] [Foundation 6][] Bootstrap 3 to Zurb Foundation 6 Style. Convert css and components from Bootstrap 3 to Foundation 6. **No Javascript!**

**[Demo][]**


## Installing
```
bower install bs3-2-zf6
```
